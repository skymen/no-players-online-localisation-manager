Translation Report

1. Select Your Language

Choose the language you want to work with from the dropdown menu.

2. Download Latest Version (Optional)

Before starting, you can download the latest version of the translation file to see what's already been translated and what needs work.

3. Upload Your Translation File

Upload your CSV translation file by either:

  • Drag & Drop: Simply drag your CSV file onto the drop zone
  • Click to Browse: Click on the drop zone to open a file picker

4. Review Translation Report

The system will analyze your file and show you:

  • Missing Terms: Terms where the translation doesn't exist
  • Missing Terms Found in Latest: Terms missing from your file but already translated in the latest data
  • Terms Needing Updates: Terms where the English text has changed compared to your file, so most likely the translation needs to be updated

Use the Diff Viewer

For terms that need updates, a detailed diff of the changes to the english text is provided.

5. Download Updated File

Once you've reviewed the report, download the updated CSV file. This file includes:

  • Your existing translations
  • Any extra translations found in the latest data
  • Up to date terms list and their english versions according to the latest data

The file will also remove any unneeded data, such as:

  • Terms that were removed from the latest data
  • Any other languages found in your file

6. Congratulations!

Once the report says everything is good, you can send the updated file to the development team for integration into the game.
